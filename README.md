# Assignment 2 - Web API
Name: Stephen Power
This project is a react-based web application that uses the TMBD API to display different movies and tv shows. 
Users can browse movies and tv shows by title, genre and sort and search. 
Detailed information listed for each movie and users can manage favourites and watchlist, and reviews. 

# Features:
- Browse movies and TV shows
- View detailed movie information (credits, providers, similar movies, reviews)
- Search, filter, and sort movies
- User authentication (sign up & login)
- Add and remove movies from a personal watchlist
- Track recently viewed movies per user
- Protected routes accessible only when logged in
- Responsive UI built using Material UI (MUI)
